Run `uv run <patternity-repo>/scripts/compile.py` from the project root (ask
the user for the patternity repo path if unknown). This is normally done
automatically by the `patternity` skill right after a pattern reaches
`proven` — use this command to force a re-sync, e.g. after manually editing
a pattern file.
